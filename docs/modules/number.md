# Number

数字表单组件

## 基本使用

``` html
<x-number v-model="value" :step="2" :min="10" :max="100"></x-number>
```

## 预览
![](./img/number.png)

## props

prop | value | default| 描述
---  |  ---  |   ---  | ---
v-model | - | - | 双向数据绑定的属性
step | Number | 1 | 步频
min | Number | 0 | 最小值
max | Number | - | 最大值
