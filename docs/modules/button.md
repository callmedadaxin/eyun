# Button

按钮组件, 各类样式可以持续添加

## 基本使用

``` html
<x-button type="primary"></x-button>
```

## 预览
![](./img/button.png)

## props

prop | value | default| 描述
---  |  ---  |   ---  | ---
type | text\primary\white\gray\orange\edit\orange-border | text | 按钮类型，具有不同样式
icon | win\ios... | - | 带图标的按钮
disabled | true/false | false | 是否禁用
